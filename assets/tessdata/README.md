# Modelos OCR de PDFlex

PDFlex incluye los modelos `spa.traineddata` y `eng.traineddata` de
[`tesseract-ocr/tessdata_best`](https://github.com/tesseract-ocr/tessdata_best).

Ese repositorio publica los modelos LSTM oficiales orientados a la mayor
precision posible. Los archivos se distribuyen bajo Apache License 2.0; la
copia de la licencia se conserva en `LICENSE`.
